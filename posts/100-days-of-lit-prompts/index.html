One Hundred Days of LIT Prompts

Learn prompt engineering through doing with new prompt patterns every weekday for 20 weeks
7 min intro video

David Colaursso
First Published January XX, 2024 (updated every weekday)

Generative AI is changing so fast it's an open question if prompt engineering will still be a thing in 20 weeks, but let's give this a go. That being said, I've always been a little susspecious of prompt engineering as an endevor. The common presentation is of the AI wisperer, crafting magical incantations. For the nearly the last 20 weeks,

Got ideas or requests for what you'd like to see? Share them with me on Mastodon @Colarusso@mastodon.social. I'll also be sure to post updates there as we go along.

Week 1

  1. Summarize and question the contents of a webpage from within the browser
  2. Use the text of an email to draft a polite reply declining any request(s)
  3. Define any selected word or phrase on a webpage
  4. Define any selected word or phrase given the context of the current webpage
  5. Fill in a "form letter" without engaging an LLM

Week 2

  1. Answer the click-bait headline
  2. Show the reciep hiding at the bottom of a webpage
  3. Find the "asks" in an email, produce a bulletted list and add to a to do list
  4. Save your to do list to a file
  5. BS with a Bot

Week 3

  1. Craft Simple Training Simulations with an "AI Actor"
  2. Diagram the selected sentence
  3. Translate and reply in original language
  4. Coin flip to poem
  5. Time of day to poem

Week 4

  1. Summarize and question the contents of a scratch pad (text area you can edit)
  2. Provide editor feedback on selected text
  3. Rewrite selected text in plain language
  4. Shorten text
  5. Expand text through dialoug with an AI writing assistant

Week 5

  1. Create a customized Summarize and question a reading
  2. Create multipe choice questions for a reading
  3. Create true-false questions for a reading
  4. Answer multiple choice questions
  5. Answer true-false questions

Week 6

  1. Answer multiple choice questions given a knowledge base
  2. Answer true-false questions given a knowledge base
  3. Answer questions based on a FAQ
  4. Answer a question based on a set of FAQs
  5. This day in history

Week 7

  1. Blurb website for social media
  2. Read a webpage and flag any logical falacies
  3. Translate angry comment into thoughtful statment in context (inject original comment anger translator)
  4. Rephrase for three audiances: 5yo, general audiance, wonk
  5. Reply to email based on a set of options

Week 8

  1. Help me write an outline
  2. Create a customized AI Word Processor. outline flesh out edit suggestions
  3. Create a customized Summ and Question for multiple papers
  4. Breakdown problem into bite-sized chunks
  5. Ability check and respond

Week 9

  1. Weekly Reflection
  2. Random simulations (diff every time)
  3. Simulations with evaluations
